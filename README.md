## 💻 Project Overview
This project is a **modern web remake** of Slash.com, focusing on:

- Smooth animations and interactive UI
- Clean layout inspired by the original site
- Fully responsive design for desktop and mobile

It’s a **practice project** to sharpen frontend development skills, including animations and responsive design.

---

## 🛠️ Tech Stack
- **React** – Component-based frontend framework  
- **Plain CSS** – Styling for responsive and clean design  
- **Framer Motion** – For smooth animations and transitions

---

## ⚡ Features
- Animated UI elements using Framer Motion  
- Responsive layout for multiple screen sizes  
- Reusable React components for sections like header, footer, and cards
